Helm Chart Registry

This repository hosts the Helm charts for Lab 8.

Available Charts:

  • my-app-chart: Includes webapp, NetworkPolicies, and Roles.

How to Use:

  1. Add this repo:
    helm repo add my-lab-repo https://zhaqsylyqq.github.io/my-helm-repo/
  2. Update:
    helm repo update
  3. Install:
    helm install my-release my-lab-repo/my-app-chart
